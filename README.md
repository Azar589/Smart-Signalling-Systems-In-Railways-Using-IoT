Project Name - Smart Signalling Systems in Railways

Abstract :

      The Smart Signalling System for Railways addresses critical visibility challenges faced by locomotive pilots, especially during adverse weather conditions such as fog and rain. The project introduces an advanced signalling solution using RF transceiver technology to ensure reliable, real-time communication between trackside signals and locomotives. RF transceivers are integrated with microcontrollers (ESP32 and Arduino Nano) to transmit signal data to a display monitor inside the train, eliminating dependency on visual cues. This system significantly enhances safety, reduces travel delays, and minimizes human error by providing clear, continuous signal visibility. With a focus on scalability, cost-effectiveness, and future IoT integration, the proposed system represents a vital step towards modernising railway infrastructure and improving transportation reliability. The system also aligns with multiple UN Sustainable Development Goals, emphasising innovation, safety, and infrastructure development.
